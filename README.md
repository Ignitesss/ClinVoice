# ClinVoice - сервис распознавания речи для медицинских консультаций

Приложение для врачей: запись **одним потоком** с микрофона; аудио накапливается для нашей дооубченной модели.
Просто говорите - наше приложение само сделаем протокол.

## Установка

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Требования к системе

- Python **3.10–3.12** (рекомендуется **3.12**)
- ~2GB disk space for Whisper models

## Как пользоваться?

1. Включите запись, говорите; при необходимости остановите и снова включите.
2. Нажмите **«Транскрибировать и заполнить протокол»** (нужно хотя бы ~1 с речи в буфере).
3. Проверьте транскрипт и протокол, при необходимости отредактируйте.
4. Сохраните протокол в **.docx** или **.txt** (или дождитесь автозагрузки `.txt`).
5. Кнопка **«Сбросить запись и черновик»** очищает буфер и черновик без транскрибации.
