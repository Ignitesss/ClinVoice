# ClinVoice - сервис распознавания речи для медицинских консультаций

Приложение для докторов, использующее распознавания речи в реальном времени. Просто говорите - приложение само напечатает.

## Установка

```bash
pip install -r requirements.txt
streamlit run app.py
```

Либо просто перейдите по ссылке, чтобы попробовать наш сервис - https://clinvoice-d6qrtxccwn8gy5zuxvnbx5.streamlit.app/

## Возможности

- Запись консультации **несколькими фрагментами** (стоп микрофона → «Добавить фрагмент» → при необходимости снова запись), затем одна кнопка транскрибации и заполнения протокола.
- Длинное суммарное аудио при распознавании **режется на части** (по умолчанию до 30 секунд на кусок), чтобы снизить нагрузку на память на хостинге.
- Скачиваемый протокол в **.docx** или **.txt**.

## Требования к системе

- Python 3.8+
- ~2GB disk space for Whisper models

## Как пользоваться?

1. Запишите фрагмент микрофоном и остановите запись.
2. Нажмите **«Добавить фрагмент в консультацию»** (при необходимости повторите для следующих частей).
3. Нажмите **«Транскрибировать и заполнить протокол»** (если последний фрагмент ещё не добавлен, он будет учтён автоматически).
4. Проверьте транскрипт и протокол, при необходимости отредактируйте.
5. Скачайте протокол в .docx или .txt.
